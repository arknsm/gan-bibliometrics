\# 代码源文件目录说明

本目录存放 GaN 文献计量项目的所有代码，按功能模块化拆分，便于维护和复用。



\## 核心模块

\- `main.py`: 项目主执行脚本（调用各模块完成完整分析流程）

\- `data\_processing.py`: 数据预处理代码

&#x20; - 功能：数据读取、清洗去重、缺失值处理、格式标准化

\- `analysis.py`: 文献计量分析代码

&#x20; - 功能：作者/机构统计、关键词分析、发文趋势计算、引用分析

\- `visualization.py`: 可视化代码

&#x20; - 功能：生成趋势图、词云、合作网络图谱

\- `utils.py`: 通用工具函数

&#x20; - 功能：文件读写、数据格式转换、日志记录



\## 运行依赖

\- 依赖库：pandas、numpy、matplotlib、wordcloud、networkx

\- 运行命令：`python main.py`
